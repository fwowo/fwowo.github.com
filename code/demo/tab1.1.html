tab1.1

约定

.kf-tab-option 默认为切换选择项

.kf-tab-pannel 默认为选择项对应面板

代码

kf.use('tab1.1', function(){ var tab = new kf.tab(Obj, Params); });

Params

eventType 触发动作，比如 mouseover | click，默认值 mouseover

index 初始显示序号，默认值 0

option 重置选择项对象，默认值 .kf-tab-option

optionClass 对象选中时样式名称，默认值 current

pannel 重置面板对象，默认值 .kf-tab-pannel

pannelClass 对象选中时样式名称，默认值 current

callback(index) 回调函数，当option被触发，并执行切换结束后被执行。

page1.0

分页

kf.use('page1.0', function(){ var page = new kf.page(Obj, Params); });

Params

model 模板代码

total 总页数，默认为 0

page 初始页码，默认为 1

hoverClass 鼠标经过时样式

currentClass 当前页码样式

callback 回调函数，当页码切换时执行，并返回当前页码。

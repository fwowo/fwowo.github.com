datepicker1.0

关于

datepicker1.0仅是对zebra-datepicker插件做了一个重新的包装，便于页面调用，及统一kf执行方法。

代码

kf.use('datepicker1.0', function(){ var datepicker = kf['datepicker']; datepicker(Obj, Params); });

Params

direction 日期区间限定，true表示今天起，常用的如：true | 1 | [1, 10] | ['2012-08-01', '2012-08-12'] | ['2012-08-01', false]

disabled_dates 日期指定限定，格式为 ['days months years weekday']，比如限定周六和周日不可选 ['* * * 0,6']

pair 使用双日期来选择日期区间时使用，在开始日期设定参数，设定结束日期对象，比如：$('#kj-date3-end')

format 设置日期展示格式，比如 'M d, Y'

show_week_number 显示周数，值为 'Wk'

view 初始界面，比如 'years'

onChange 日期界面改变时触发 function(view, elements){}

always_visible 将日期控件固定在某对象中，比如 $('#dateplace')

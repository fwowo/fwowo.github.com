overlay1.1

代码

kf.use('overlay1.1', function(){ var overlay = new kf.overlay(Params); });

Params

width 弹出框宽度，默认值 'auto'

height 弹出框高度，默认值 'auto'

closeBtn 是否有关闭按钮，默认值 true

headContent 弹出框head内容，默认值 null

bodyContent 弹出框body内容，默认值 null

footContent 弹出框foot内容，默认值 null

mask 是否有半透明背景，默认值 true

maskClose 点击背景是否触发关闭，默认值 true

showCallback 提出框展现时回调函数

closeCallback 提出框关闭时回调函数

调用函数

content 重置弹出框内容

show 显示弹出框

hide 隐藏提出框

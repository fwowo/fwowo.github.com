ckplayer1.0

关于

ckplayer1.0仅是对ckplayer 6.0 播放器做了一个重新的包装，便于页面调用，及统一kf执行方法。

代码

kf.use('ckplayer1.0', function(){ var ckplayer = kf['ckplayer']; ckplayer(Obj, Params); });

Params

width 播放器宽度，默认 600

height 播放器高度，默认 400

image 播放器初始图片

file 播放文件

autostart 是否自动播放，默认 true

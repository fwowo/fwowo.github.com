waterfall2.0

约定

.kf-waterfall-column 默认为瀑布流列内容容器

代码

kf.use('waterfall2.0', function(){ var waterfall = new kf.waterfall(Obj, Params); });

Params

column 瀑布流列内容容器，默认值 .kf-waterfall-column

model 模板代码

data 模板数据

loadimg 需加载图片字段

callback 回调函数，内容全部展示完毕后执行。

调用函数

pause 暂停加载展示

play 继续加载展示

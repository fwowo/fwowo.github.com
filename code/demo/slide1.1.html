slide1.1

约定

.kf-slide-wrap 默认为滚动元素容器

.kf-slide-content 默认为滚动元素内容容器

.kf-slide-pannel 默认为滚动元素内容

若.kf-slide-content不存在，.kf-slide-wrap下的ul替之，若.kf-slide-pannel不存在，.kf-slide-wrap下的li替之。

代码

kf.use('slide1.1', function(){ var slide = new kf.slide(Obj, Params); });

Params

effect 切换效果，比如 horizontal | vertical | fade，默认值 horizontal

autoPlay 是否自动播放，默认值 true

autoTime 自动切换时间差，单位毫秒，默认值 5000

autoRound 是否循环，默认值 true

iconState 是否展示小图标，默认值 true

iconEvent 小图标触发事件，比如 mouseover | click，默认值 mouseover

zebra-datepicker / 时间控件

zebra-datepicker是基于jQuery的时间控件，功能很强大，除了基本功能外，还有一个年度第几周显示功能，此功能深得本人喜爱。它的界面也很简洁端庄，包含两款皮肤，都很美观。以下是两款皮肤的css和js引用地址：

<link rel="stylesheet" href="http://fwowo.github.com/static/zebra-datepicker/css/zebra_datepicker.css" /> <link rel="stylesheet" href="http://fwowo.github.com/static/zebra-datepicker/css/zebra_datepicker_metallic.css" /> <script src="http://fwowo.github.com/static/zebra-datepicker/zebra_datepicker_cn.js"></script>
普通模式：
$('#J-date1').Zebra_DatePicker();
日期限定：
$('#J-date2').Zebra_DatePicker({ direction: true, // true or 1 or [1, 10] or ['2012-08-01', '2012-08-12'] or ['2012-08-01', false] disabled_dates: ['* * * 0,6'] // days months years weekday });
日期周期： -
$('#J-date3-start').Zebra_DatePicker({ direction: true, pair: $('#J-date3-end') }); $('#J-date3-end').Zebra_DatePicker({ direction: true });
日期格式：
$('#J-date4').Zebra_DatePicker({ format: 'M d, Y' });
周数：
$('#J-date5').Zebra_DatePicker({ show_week_number: 'Wk' });
初始界面：
$('#J-date6').Zebra_DatePicker({ view: 'years' });
change交互： （view切换时触发）
$('#J-date7').Zebra_DatePicker({ onChange: function(view, elements) { console.log(view); console.log(elements); } });
固定位置：
$('#J-date8').Zebra_DatePicker({ always_visible: $('#dateplace') });
